LoRA with ChatGPT Bot

What is a LoRA?

LoRA (Low-Rank Adaption of Large Language Models) is a training technique used to make deep learning models more useful for specific tasks. In context of image generation, LoRA models can be applied ontop of base model to achieve certain results. For example, you can generate certain people in certain clothes or poses, as well as different characters.

How do I use LoRA's with the bot?

ChatGPT Bot allows you to use LoRA models that
were uploaded to CivitAI.
Enter the link to the LoRA you want to use into the appropriate field of the /imagine slash command:

Your chosen model must meet the following requirements, otherwise it will be ignored:

· Support .safetensors format
· Weight less than 150mb
